Lesson n:

Everything you need to run JavaScript comes built into your web browser. This lesson will show you how to get started.

Just follow this proceedure step-by-step:

  1. Start the Firefox web browser. On a current Ubuntu system, you should see a window that looks something like this:
  2. Right click with your mouse in any blank space on the page, and click on Inspect Element (Q) (Your screen should look something like this just before you click).
  3. Click on the > Console link toward the upper-left of the new web development console at the bottom of your browser window. (Look at the mouse pointer in this screen shot to see where to click). Next click on the Logging (R) button on the 2nd row of buttons that appear after clicking > Console
  4. Move your mouse over toward the upper-right of the web console and click on the icon that looks like a small spiral bound notebook. If you hover with your mouse over this icon it will say Scratchpad (Look at the mouse pointer in this screen shot to see where to click).
  5. After clicking on the Scratchpad icon a new window will open and your desktop should look something like this:
  6. The Scratchpad window is a simple JavaScript IDE that enables you to load JavaScript files, edit them, save them, and run them. Type the following line of JavaScript code into the Scratchpad window: 
      console.log("Hello, World!");
    After you type this, click on the Run button, and you should see the text Hello, World! appear in the Logging (R) screen. A side-by-side view of the two windows after you are finished should look something like this:
