Lesson 3 - Remove Files

Since you have already learned how to add files with bzr add its time to learn about removing files. Say you want to remove test3.txt from the repository

bzr remove --force test3.txt

This will now delete the file from the repository. If you tried removing the file without bzr (the same way you would remove any other file on your computer) then you would have removed the file, but it would not be deleted from the repository.

deleted test3.txt

This is what will come up after forcing the delete. If you deleted the wrong file you can revert to get the file back.
