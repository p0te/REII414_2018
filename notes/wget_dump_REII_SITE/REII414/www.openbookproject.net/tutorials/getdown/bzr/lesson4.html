Lesson 4 - Plugins

bzr has plugins that can be added to customize it. Launchpad, an intergration plugin, is one example of a bzr plugin. There are also command plugins such as bzr email which sends a message after a commit to the bzr commits mailing list.

You can go the bzr website and look at the list of plugins and learn how to install them.

If you wanted to see what plugins you have:

bzr plugins

Typing this in will allow you to see the the plugin and the description of the plugin.

launchpad 
Launchpad.net integration plugin for Bazaar.
