Lesson 2: Change Tools

Now let's explore the next set of tools. These are what we like referring to as the change tools.

Change Tools

The first of these tools is the alignment tool.

The Alignment Tool

It's base purpose is to align text around a given image. When the tool is selected the following menu is brought up.

The Alignment Menu

There are two different ways to align. The first method is called Relative To:. This aligns the object that needs to be aligned around a selected object. There are two sets of control keys. The first set of keys corrosponds with Relative To.

Look at where it says First Item. This is a drop down menu. This allows the user to align the item that needs aligning to be aligned with the selections. The options are the following:

Relative To Dropdown Menu

The next set is called Distribute. The controls are relatively the same, except the function is different. The Relative to will align around a certain object, but the distribute will align around the entire image.

The next tool is called the crop tool

Crop Tool

The purpose of this tool is to take an image and cut out a piece of that image and make a new image from it. *Note this is different then scaling the image because it will cut out the unselected parts. Observe.

The next tool is called the rotate tool.

Rotate Tool

The purpose of this tool is to rotate the image via its axis. This is the menu for it.

Rotate Menu

First click on the rotate tool in the GIMP menu. Then go to the layer that is meant to be edited and click on it. Then click the actual picture. Start by clicking any of the corners and move the mouse so that the image is rotated as much as you want. The proccess should vaguely look like this.

Show Rotate

The other way of rotating is by using the rotate menu.

Rotate Settings

This has some advantages as well as disadvantages. The advantages are, first, the image can be rotated numerically. The negative numbers rotate counterclockwise and the positive numbers rotate clockwise. The scroll bar can also be used for rotation. The other advantage is that the point of origin (the place where image pivots) can be changed using the pixel numbers. However, if the user doesn't know much about angles and what pixels are, that feature is relatively unuseful.

The next tool is called the scale tool. Its purpose is to change the size of the actual image. It differs from the crop tool because it does not select a region, instead it expands or shrinks the entire image.

Scale Tool

To start the scaling process, first start by opening the image. Then click on the image window in th GIMP. This should promt the scaling window. Make sure the right layer is selected.

Scale Menu

This menu allows for the user to scale and keep the proportions the same. The other method of scaling is to click on one of the corners of the image and drag the mouse to the appropriate size.

Scale Free Hand

When expanding it might be useful to zoom out. To do so hit z on the keyboard or click on the tool in the tool menu that looks like a mangifying glass. That should prompt the zoom menu.

Zoom Menu

Click zoom out and click on the image once this should zoom out once. This allows the user to see the entire window. This is convenient because now the real size of the layer can be seen.
